# CHLiving

Conectando vecinos, fortaleciendo comunidades.

## Descripción

CHLiving es una plataforma digital para condominios y comunidades residenciales enfocada en:

- Marketplace vecinal.
- Avisos y comunicados.
- Reservas de amenidades.
- Reportes.
- Directorio comunitario.

## Filosofía

CHLiving no busca ser un marketplace tradicional.

Su objetivo principal es fortalecer la convivencia vecinal y apoyar a los residentes que ofrecen productos o servicios.

## Funcionalidades

### Marketplace Vecinal
- Venta entre vecinos.
- Emprendimientos.
- Donaciones.
- Intercambios.

### Avisos
- Comunicados.
- Eventos.
- Asambleas.

### Reservas
- Salón de eventos.
- Roof Garden.
- Áreas comunes.

### Reportes
- Mantenimiento.
- Seguridad.
- Limpieza.
